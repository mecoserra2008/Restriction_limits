# 📘 **Excel Dashboard Template (Ready to Build)**

Below is the full template structure.  
Create **two sheets**:

---

# 🟦 **Sheet 1 — “Data”**  
Paste your raw data here with these exact column headers:

| Contraparte | Tipo de linha | Limite | Utilizado | % utilização | País | Tipo | Numeração Rating Basileia | Limites RAF globais | Limites RAF Individual |
|-------------|---------------|--------|-----------|--------------|------|------|----------------------------|----------------------|-------------------------|

Add the following **calculated columns** to the right:

### **Column L — Available_Limit**
```
=[@Limite] - [@Utilizado]
```

### **Column M — Effective_RAF**
```
=MIN([@[Limites RAF globais]]; [@[Limites RAF Individual]])
```

### **Column N — Final_Capacity**
```
=MIN([@Available_Limit]; [@Effective_RAF])
```

Format these columns as **Number**, 2 decimals, thousand separators.

## 📊 **2. Pivot Table — Capacity Summary**

Insert a Pivot Table from the **Data** sheet with the following layout:

### **Rows**
- Contraparte  
- Tipo de linha  
- País  

### **Values**
- **Final_Capacity** → *Min*  
- **Available_Limit** → *Min*  
- **Effective_RAF** → *Min*  
- **Utilizado** → *Sum*  
- **Limite** → *Sum*  

Rename the fields:

| Pivot Field | Display Name |
|-------------|--------------|
| Min of Final_Capacity | Final Capacity |
| Min of Available_Limit | Min Available |
| Min of Effective_RAF | Min RAF |
| Sum of Utilizado | Total Used |
| Sum of Limite | Total Limit |

Format all values as **#,##0**.

---

## 🚦 **3. Conditional Formatting**

Apply to the **Final Capacity** column:

### **Rule 1 — Red (Critical)**
If:
```
Cell Value < 0
```
Format: **Red fill, white bold text**

### **Rule 2 — Orange (Warning)**
If:
```
Cell Value < 0.2 * [Total Limit]
```
Use a formula:
```
=F2 < 0.2 * J2
```
(Adjust column letters based on your pivot layout)

Format: **Orange fill, black bold text**

### **Rule 3 — Green (Healthy)**
If:
```
Cell Value >= 0.2 * [Total Limit]
```
Format: **Green fill, white text**

---

## 📈 **4. Visuals (Charts)**

### **Chart 1 — Remaining Capacity by Counterparty**
- Use **Final Capacity**
- Clustered bar chart
- Sort descending
- Title: **"Available Capacity by Counterparty"**

### **Chart 2 — Utilization vs Limit**
- Use **Total Used** and **Total Limit**
- Combo chart:  
  - Columns = Total Limit  
  - Line = Total Used  
- Title: **"Utilization vs Limit"**

### **Chart 3 — RAF Constraint Impact**
- Use **Min RAF** vs **Min Available**
- 100% stacked bar chart
- Title: **"Impact of RAF Restrictions"**

---

## 🧭 **5. KPI Cards (Top of Dashboard)**

Insert 3 large text boxes:

### **KPI 1 — Total Final Capacity**
Formula:
```
=SUM(Data!N:N)
```

### **KPI 2 — Total Utilizado**
```
=SUM(Data!G:G)
```

### **KPI 3 — Total Limit**
```
=SUM(Data!F:F)
```

Format each as:

- Font: Segoe UI Semibold, 24 pt  
- Background: Light gray  
- Border: Thin  
- Alignment: Center
